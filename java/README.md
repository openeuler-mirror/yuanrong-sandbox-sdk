# openYuanrong Sandbox SDK for Java

Planned language SDK directory. Keep future Maven/Gradle project files,
examples, and tests here, for example:

```text
java/
  pom.xml or build.gradle
  src/main/java/
  src/test/java/
  examples/
```

The Java SDK should follow the repository-wide concepts in the root README and
use the same environment variables and route semantics.
